## How to Play

1. Visit https://jbg9696.github.io/who-are-you-today/
2. Answer 5 questions about your current preferences and feelings
3. Discover which personality type matches you today
4. Share your results and come back tomorrow for a new quiz!

## Technical Details

This project uses:
- HTML5

- Lucide Icons

## Local Development

To run this project locally:
